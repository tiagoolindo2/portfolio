Meus projetos

Indicando musica: Uma Webpage indicando musica boa

Este projeto é uma página web que apresenta uma lista de musica favoritos, incluindo informações sobre os autores, datas de publicação e links para compra na Amazon. A página é estilizada com CSS para uma visualização agradável e usa fontes externas do Google Fonts.

Venda de curso: Um site de venda do curso do alura

Esse projeto é um site de venda de curso uma plataforma online onde os usuários podem comprar acesso a cursos digitais, que podem ser em vídeo, texto, áudio ou uma combinação desses formatos. O objetivo principal do site é facilitar a divulgação, venda e distribuição dos conteúdos educativos, permitindo que os alunos aprendam de forma prática e acessível. p>

Jogo do pong: Uma Aventura com seu amigo com o jogo do pong com Scratch

Este projeto consiste na criação do clássico jogo Pong, um dos primeiros videogames da história, que simula uma partida simples de tênis de mesa digital. O objetivo principal foi desenvolver um jogo interativo e funcional, com controles básicos, que permita a dois jogadores competirem entre si.
